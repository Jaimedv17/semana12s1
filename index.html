1.Como se transfiere la informacion basado en el protocolo http

Primero se genera el Request para obtener un Response y todo viaja o debe de viajar en formato Json

2.Que son los verbos HTTP y cuales usamos

Los vervos son como acciones que realizaran, como en el caso de GET que solicita informacion. Vervos mas usados

VERVOS: Get, Post, Put, Delete

3.Que es un crud

Estas operaciones forman la base de la funcionalidad en sistemas que interactúan con bases de datos.

Create (Crear): Crear nuevos datos o registros en la base de datos.

Read (Leer): Leer o consultar datos existentes en la base de datos.

Update (Actualizar): Actualizar o modificar datos existentes en la base de datos.

Delete (Eliminar): Eliminar datos o registros existentes en la base de datos.

4.Que es asincronia y sincronia

La asincronia es la forma en la que permite cargar los objetos estaticos primero, y se añaden

La sincronia es solo objetos estaticos que cargarn al instante.

6.Como realizar peticionas js / como manejar la data

Se pueden realizar peticiones mediante xmlHTTPRequest, fethc y axios (cabe recalcar que axios es una)

La data se maneja por medio de Callback, Promises y Async/Await.
